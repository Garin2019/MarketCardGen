# 📁 Папка для пользовательских шрифтов

## 🎯 Назначение

Эта папка предназначена для хранения пользовательских шрифтов локально в проекте.

## 📥 Как добавить шрифт

### Вариант 1: Вручную скопировать файл

1. Скачайте TTF/OTF/WOFF/WOFF2 файл шрифта
2. Скопируйте в эту папку: `public/fonts/`
3. Перезапустите приложение

### Вариант 2: Через UI приложения

1. Откройте приложение
2. Создайте шаблон → Выберите блок
3. В секции "Шрифт" → "📁 Выбрать файл"
4. Загрузите шрифт
5. Шрифт сохранится в localStorage

## 📂 Структура

```
public/
└── fonts/
    ├── MyFont.ttf           ← Ваши шрифты здесь
    ├── AnotherFont.woff2
    ├── CustomFont.otf
    └── README.md            ← Этот файл
```

## 🔧 Использование локальных шрифтов

### Способ А: Автоматическая загрузка всех шрифтов из папки

См. файл: `LOCAL_FONTS_GUIDE.md` в корне проекта

### Способ Б: Регистрация шрифтов в CSS

Добавьте в `src/index.css`:

```css
@font-face {
  font-family: 'MyFont';
  src: url('/fonts/MyFont.ttf') format('truetype');
  font-weight: normal;
  font-style: normal;
}

@font-face {
  font-family: 'AnotherFont';
  src: url('/fonts/AnotherFont.woff2') format('woff2');
  font-weight: normal;
  font-style: normal;
}
```

Затем шрифты будут доступны в выпадающем списке!

## ⚠️ Важно

- ✅ **Максимальный размер файла:** Без ограничений (локальное хранение)
- ✅ **Форматы:** TTF, OTF, WOFF, WOFF2
- ✅ **Синхронизация:** Автоматически включаются в Git
- ✅ **Резервные копии:** Сохраняются вместе с проектом

## 💡 Рекомендации

1. **Используйте WOFF2** — самый компактный формат
2. **Называйте файлы понятно:** `Roboto-Bold.woff2`, а не `font1.ttf`
3. **Не храните дубликаты:** Один шрифт = один файл
4. **Проверяйте лицензии:** Убедитесь что можно распространять
